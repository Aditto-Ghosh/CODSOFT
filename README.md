# CODSOFT
# 🎬 Movie Genre Prediction using Machine Learning

This project predicts the genre of a movie based on its plot summary using Machine Learning techniques.

---

## 🚀 Features
- Predicts movie genre from text input
- Uses TF-IDF vectorization
- Trained ML model (Naive Bayes / Logistic Regression)
- Simple web interface using Flask

---

## 🛠️ Tech Stack
- Python
- Scikit-learn
- Flask
- HTML/CSS
